## What is the Music App?

The **Music App** is a web application that serves as a Spotify clone. It allows users to create an account, create playlists and add songs to them, and view songs, albums, and artists.

### Key Features:

- **Search For Songs, Artists, and Albums From the Database**:  
- **Create Playlists**: Keep track of your media collection by adding them to your personal "shelf".
- **User Authentication & Authorization**: Secure user login and account management for a personalized experience.
